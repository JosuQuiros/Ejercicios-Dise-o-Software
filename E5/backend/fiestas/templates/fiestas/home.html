Monolito Django con capas modelo → repositorio → servicio → vista.

API (JSON)

  • GET /fiestas — listar fiestas con cupo disponible
  • POST /fiestas — crear fiesta (nombre, ubicacion, capacidad, opcional latitud/longitud/descripcion)
  • GET /fiestas/ubicaciones — todas las fiestas (para mapa)
  • GET /fiestas/{id}/invitaciones/pendientes
  • POST /fiestas/{id}/invitaciones — solicitud con nombre_invitado
  • POST /fiestas/{id}/invitaciones/{id}/aceptar
